Instructivo SAO

Instrucciones de Uso

Optimizador > Optimizador

Nombre:

Denominación descriptiva para el Plano que se va a optimizar. Este nombre se persistirá en la base datos, en la parte de “Optimizador > Historial de Optimización”, explicada mas adelante, siempre y cuando se marque el checkbox “Registrar en Historial”. Permitirá buscar los planos optimizados en el historial con este nombre.

Cantidad de personas:

Se debe ingresar la cantidad de personas que se estima que trabajaran en la oficina. Este parámetro se utiliza para calcular el ancho de los pasillos, basado en el cálculo del código de construcción de la ciudad de Buenos Aires, art. 1º pto Anexo de la Ley Nº962 BOCBA 1607 del 13/01/2003.

Escala:

Escala del plano ingresado. La escala se utiliza para ajustar la escala de los muebles a la del plano.

Registrar en Historial:

Si se marca este checkbox, se persistirá en base de datos los parámetros y los planos generados optimizados. El historial se puede consultar en la opción del menú “Optimizador  Historial de Optimización”.

Optimizar costo:

Si se marca este checkbox, se habilita el campo “Costo máximo” para ingresar un presupuesto. En la parte derecha de la pantalla se muestra el costo calculado, sumatoria del costo de todos los muebles que se ingresen en la lista de Muebles, en base al presupuesto ingresado en “Costo máximo”, el costo se mostrará en verde si no se superó el “Costo máximo” o rojo si se superó.

Seleccionar plano:

Este botón abre una ventana para elegir el plano de la oficina a optimizar. El plano debe residir en el directorio que se seteó en el archivo de configuración, o un directorio interno de este.

Mueble:

Lista desplegable que permite elegir un mueble de la base de datos para ingresarlo en la lista de muebles que se desean ubicar en la oficina.

Cantidad:

Debe ingresarse la cantidad de muebles de el tipo ingresado en “Mueble” que se desea ubicar en la oficina.

Agregar:

Inserta el mueble elegido, con la cantidad indicada, en la lista de muebles para optimizar. La lista se irá actualizando y mostrando debajo, con las columnas:

  • Mueble
  • Cantidad
  • Eliminar

Generar Resultado:

Ejecuta el proceso de optimización de muebles en la oficina, generando los planos optimizados. Se ofrecerán dos opciones de planos optimizados, uno con los pasillos en largo, otro con los pasillos en ancho. Ambos planos quedarán guardados en el directorio previamente seteado en el archivo de configuración.

Optimizador > Historial de Optimización

Nombre:

Con este parámetro se buscará en la base de datos el plano optimizado deseado, debe coincidir con el nombre con el que se grabó previamente. Si se deja vacío, se devuelve el historial completo.

Buscar:

Busca en el historial los planos optimizados, utilizando el parámetro “Nombre” y devuelve una lista con los resultados:

  • Nombre
  • Cantidad de personas
  • Escala
  • Costo máximo
  • Visualizar/Eliminar
Al hacer click en el botón “Visualizar” con forma de lupa, se obtienen los planos optimizados correspondientes al historial. Los planos quedan grabados en el directorio previamente seteado en el archivo de configuración.

Configuración > Mueble

Nombre:

Ingresar el nombre del mueble con el q se desea trabajar.

Código:

Código numérico único para cada mueble.

Agregar:

Lleva a la pantalla de “Alta de muebles” Campos para alta:

  • Nombre
  • Código
  • Largo (En metros)
  • Ancho (En metros)
  • Precio (En pesos)
  • Distancia mínima al prox. Mueble (en metros)
  • Distancia a las paredes
  • Orden de prioridad
  • Posee Radio (Habilita campos para cargar radio, si fuera un mueble redondo)
  • Seleccionar Imagen (Permite cargar archivo JPG con imagen del mueble)

Buscar:

Utiliza los parámetros “Nombre y “Código” para seleccionar los muebles de la base de datos que coincidan. Muestra una lista de muebles que coinciden:

  • Imagen
  • Mueble
  • Código
  • Estado
  • Visualizar/Editar/Eliminar
Al hacer click en el botón “Editar”, con forma de lápiz, lleva a la pantalla “Modificación de muebles” que permite editar los parámetros de ese mueble y grabar. Los campos editables, son los mismos habilitados en “Alta de Muebles”.

Seguridad > Usuarios

Nombre:

Ingresar el nombre del usuario con el q se desea trabajar.

Apellido:

Ingresar el apellido del usuario con el q se desea trabajar.

Usuario:

Ingresar usuario con el q se desea trabajar.

Rol:

Seleccionar rol con el q se desea trabajar.

Estado:

Seleccionar estado con el q se desea trabajar.

Agregar:

Lleva a la pantalla de “Alta de usuarios” Campos para alta:

  • Nombre
  • Apellido
  • Usuario
  • E-Mail
  • Rol
  • Activo

Buscar:

Utiliza los parámetros “Nombre, “Apellido”, “Usuario”, “Rol” y “Estado” para seleccionar los usuarios de la base de datos que coincidan. Muestra una lista de usuarios que coinciden:

  • Id
  • Nombre
  • Apellido
  • Usuario
  • Rol
  • Activo
  • Visualizar/Editar/Eliminar
Al hacer click en el botón “Editar”, con forma de lápiz, lleva a la pantalla “Modificación de usuarios” que permite editar los parámetros de ese usuario y grabar. Los campos editables, son los mismos habilitados en “Alta de Usuarios”.

Seguridad > Roles

Nombre:

Ingresar el nombre del rol con el q se desea trabajar.

Estado:

Seleccionar estado con el q se desea trabajar.

Agregar:

Lleva a la pantalla “Alta de Roles.” Muestra la lista de reglas para seleccionar cuales aplican al nuevo rol.

  • Regla
  • Activo (Checkbox: Aplica/No aplica)

Guardar:

Guarda el rol en la base de datos.

Guardar y Agregar otro:

Guarda el rol en la base de datos y permite seguir agregando roles.

Buscar:

Utiliza los parámetros “Nombre y “Estado” para seleccionar los roles de la base de datos que coincidan. Muestra una lista de roles que coinciden:

  • Id
  • Nombre
  • Activo
  • Visualizar/Editar/Eliminar
Al hacer click en el botón “Editar”, con forma de lápiz, lleva a la pantalla “Modificación de roles” que permite editar las reglas de ese rol y grabar. La pantalla tiene el mismo layout que “Alta de Roles”.

Seguridad > Log de usuarios

Usuario:

Usuario para consultar el log.

Buscar:

Utiliza el parámetro “Usuario” para seleccionar los logs de la base de datos de ese usuario. Muestra una lista:

  • Usuario
  • Descripción (Acción realizada por el usuario en el sistema)
  • Fecha
  • Eliminar

Seguridad > Cambiar password

Permite al usuario logueado cambiar su password.

  • Contraseña actual
  • Nueva contraseña
  • Repetir Nueva contraseña
